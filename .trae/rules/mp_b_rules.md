# 🐾 B 端小程序（服务人员端）业务愿景与功能规则

本规则面向 **B 端服务人员**（宠托师/洗护师/遛狗师等），用于从业务角度约束与指导小程序的功能范围、交互重点与增长方向。

## 1. 产品愿景
让任何具备专业能力与责任心的服务人员，都能用一部手机完成「接单—履约—结算—复盘」的闭环，稳定获得订单与口碑增长。

### 1.1 价值主张
- **对服务人员**：更稳定的单量、更可控的时间安排、更清晰的收入结构、更快的复购与转介绍。
- **对平台**：更高履约率、更低客服介入、更强服务标准化与可复制性。
- **对用户（间接）**：更可靠的服务体验、更透明的过程与更一致的质量。

## 2. 核心用户画像
### 2.1 新入驻服务人员
目标：快速完成资质与档期建设，尽快获得首单与正向评价。

### 2.2 稳定接单服务人员
目标：提升接单效率与履约质量，减少沟通成本，最大化有效工时与收入。

### 2.3 头部服务人员（品牌化）
目标：打造个人品牌与差异化服务，维护高复购客户群，减少低质量订单。

## 3. 业务闭环与关键路径
### 3.1 从 0 到 1（入驻）
1) 入驻申请 → 2) 资质审核 → 3) 个人资料与服务项完善 → 4) 价格与服务半径设置 → 5) 档期可用性设置 → 6) 首单引导。

### 3.2 从单到现金（履约）
1) 订单通知 → 2) 接单/拒单（含原因） → 3) 出发/到达 → 4) 服务中过程记录 → 5) 服务完成 → 6) 费用确认（含加项/差旅） → 7) 评价与申诉 → 8) 结算入账。

### 3.3 从现金到增长（复盘）
1) 数据看板（接单率/履约率/评分） → 2) 服务能力补齐（模板化建议） → 3) 客户维护（复购提醒/回访） → 4) 个体品牌资产沉淀（作品与案例）。

## 4. 功能域划分（B 端必须聚焦）
### 4.1 入驻与合规
- 实名/资质/健康证等材料提交与更新
- 审核状态追踪与补充材料提醒
- 合规提示：服务边界、安全、隐私、上门风险

### 4.2 个人主页与服务配置
- 个人资料：头像、昵称、城市、标签、擅长宠种
- 服务项：上门洗护/上门喂猫/遛狗（可扩展）
- 价格体系：基础价 + 可选加项 + 远距离/夜间/节假日规则
- 服务半径：地理范围与可达时间
- 档期管理：可接单时段、休息日、临时锁档

### 4.3 订单中心（B 端的“操作台”）
- 订单状态：待接单/待出发/服务中/待确认/已完成/售后
- 接单能力：批量确认、超时策略、拒单原因结构化
- 履约工具：一键导航、到达打卡、过程记录模板
- 异常处置：迟到、用户改期、地址不符、宠物不配合、无法进门等

### 4.4 履约过程记录（降低争议）
- 关键节点：出发、到达、服务开始、服务结束
- 过程凭证：图片/短视频/文本（按服务类型模板化）
- 用户沟通：高频话术与快捷回复（减少自由输入成本）

### 4.5 收入结算与财务
- 钱包：可提现余额、待入账、冻结/争议款
- 订单结算明细：基础服务费、加项、平台服务费、补贴
- 提现：银行卡/规则提示/到账时间

### 4.6 评价与售后
- 评价：评分与标签、可申诉入口、举证上传
- 纠纷：规则说明、处理进度、平台介入机制

### 4.7 成长与工具（中长期）
- 数据看板：接单率、履约率、超时率、近 30 天收入、复购率
- 成长任务：完善资料、首单、连续履约、低投诉
- 运营活动：时段补贴、专题服务、冲榜激励

## 5. B 端业务约束（Do / Don’t）
### 5.1 必须坚持
- **效率优先**：B 端每个页面都应围绕“更快接单/更稳履约/更少争议”。
- **模板化**：高频流程提供模板与结构化填写，避免让服务人员写“小作文”。
- **可追溯**：关键动作必须可回溯（时间点、凭证、状态流转）。
- **弱打扰**：通知要分级（强提醒/弱提醒/静默），避免信息轰炸。

### 5.2 避免发生
- 把 B 端做成“信息展示端”而不是“作业端”。
- 用复杂表单堆砌能力，而不提供可用的默认值与引导。
- 关键状态没有明确下一步动作（用户不知道此刻该做什么）。

## 6. 服务类型差异（用于流程与模板设计）
### 6.1 上门喂猫
- 关键价值：稳定、细致、可视化（铲屎、添粮、换水、陪玩）
- 争议点：是否到场、是否按要求操作、是否与多猫混淆

### 6.2 遛狗
- 关键价值：安全、准时、路线与时长
- 争议点：时长不足、牵引与突发、排泄与清洁

### 6.3 上门洗护
- 关键价值：专业度、耗时可预期、加项透明
- 争议点：加项收费、毛发打结处理、宠物应激风险

## 7. 关键指标（B 端产品成败衡量）
- 接单率、超时率、履约率
- 纠纷率、申诉成功率、客服介入率
- 平均订单耗时（含沟通成本）、复购率
- 近 30 日收入稳定性（波动幅度）

## 8. 未来扩展方向（设计时预留）
- 多服务项组合套餐（洗护 + 陪护）
- 团队/机构模式（多服务人员协作、排班）
- 资质分级与差异化展示（认证、徽章）
